Top Command in Linux/Unix

Overview

The top command displays all the running processes on your system, helping monitor usage and performance. It is widely used by system administrators to detect server load and ensure efficient resource utilization.

“top” stands for Table of Processes.

What Does Top Command Do?

The top command generates an ordered list of active processes. The list updates periodically and can be customized extensively to suit your requirements.

It provides vital information about:

  • Memory and CPU utilization
  • Processes and users consuming resources
  • Load averages and more

Implementations

Several implementations of the top command exist:

  • Unix Version: Created by William LeFebvre and later hosted on SourceForge.
  • Linux Version: Part of the procps-ng tools, initially developed by Roger Binns.
  • Windows Equivalent: Task Manager utility and tasklist command.
  • Solaris Equivalent: prstat command.
  • IBM AIX: Active processes listed via topas and topas_nmon.

Options of Top Command

Option Description
-a Sorts processes by allocated memory.
-b Starts top in batch mode for output redirection.
-c Reverses the last remembered state of command line programs.
-d Displays delay between screen updates.
-h Shows usage and library version, then quits.
-H Reverses the remembered thread state.
-i Excludes idle processes from the list.
-m Reports memory usage instead of virtual memory.
-M Displays memory in floating point units.
-n Limits the number of iterations before quitting.
-p Monitors processes by specific PIDs.
-s Runs securely without certain interactive commands.
-S Displays cumulative CPU time.
-U Filters processes by a specific user.

Example of Top Command

Here's an example of using the top command in the terminal:

$ top

Running the command without any arguments displays a dynamic list of processes, including their resource usage (CPU, memory), and additional system performance metrics.

For more advanced usage, you can customize the display or change the sorting method. For example:

$ top -o %CPU

This command will sort processes by CPU usage, showing the most resource-intensive ones at the top.

If you want to quit the `top` command, you can simply press the q key to exit the view.
